SublimeLinter-contrib-jade
-------------------------------
This linter plugin for SublimeLinter provides an interface to jade.

** IMPORTANT! **

Before this plugin will activate, you *must*
follow the installation instructions here:

https://github.com/zekesonxx/SublimeLinter-contrib-jade

Additionally, be aware that this plugin does NOT work realistically, if using it on more than single file it starts to have issues.
